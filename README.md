Slate for Apitome.

## Usage

How to use my plugin.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'apitome_slate'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install apitome_slate
```
